ComfyUI楼层生图
这个插件会给 AI 消息楼层右侧加一个生图按钮。点击后会从该楼层提取提示词，调用你本地已经配好的 ComfyUI 工作流，再把结果图片插入酒馆输入框。
启用插件
仅 AI 消息显示按钮
自动后台生图
ComfyUI 地址
提示词提取正则
默认优先支持固定格式：[ImagePrompt|绘图提示词]，也兼容 <ImagePrompt>...</ImagePrompt>、【绘图提示词】...、绘图提示词: ...
给 AI 的固定出图规则
把这段规则粘到角色卡、系统提示词或世界书里，让 AI 每次正文结尾都自动输出可提取的绘图提示词。
工作流 JSON
现在支持直接粘贴 ComfyUI 前端保存的 workflow JSON，也兼容 API prompt JSON；也可以上传本地 .json / .txt 文件，插件会读取文件文本填入这里。
文件名前缀模板
可用变量：${timestamp}、${slug}、${rand}
固定 Seed（可空）
设置迁移
导出/导入所有设置（ComfyUI地址、工作流、正则等），方便迁移到其他电脑

x
放大倍率
仅当工作流里存在 ImageScale 等放大节点时才会自动写入放大后的宽高；如果没有放大节点，将直接跳过，不影响正常出图。
